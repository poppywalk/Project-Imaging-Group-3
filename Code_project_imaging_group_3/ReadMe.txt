CNN_model.py:
Contains the code for the CNN_model that is used to classify the images.

GAN_model_group_3.py:
Contains the code for the training of the GAN model. It also contains code to save the losses and plot the generated images.

GenerateImages.py:
Is the code used to create the new dataset of images.

Submission_Small_Model.csv:
Is the kaggle submission file for the CNN_model without the extra set of generated images.

Submission_Big_Model.csv:
Is the kaggle submission file for the CNN_model with the extra set of generated images.

Generator_model_final_075_final:
Is the trained model with which the images were made.
